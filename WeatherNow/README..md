# Weather Now

# Weather Now is a responsive weather application built with React (TypeScript) and Tailwind CSS.  
It provides real-time weather updates, forecasts, and location-based data using the Open-Meteo API.

---

# Live Demo
https://codesandbox.io/p/sandbox/f4683p
---

# Features
- Search weather by city name
- Detect weather for your current location
- View 6-day forecast with icons and animations
- Real-time local date and time
- Light / Dark mode toggle (saved in localStorage)
- Previously searched cities stored locally
- Fully responsive layout for all devices

---

# Tech Stack

 React + TypeScript -  Core framework 
 Tailwind CSS       -  Styling and responsiveness 
 Framer Motion      -  Animations and transitions 
 Lucide Icons       -  Modern icon set 
 Open-Meteo API     -  Weather and geocoding data
